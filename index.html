HTMAPL: HyperText Geographic Maps for jQuery

Stamen

HTMAPL is:

  1. An HTML vocabulary for describing both interactive and static geographic maps with layers and markers
  2. A Polymaps-inspired—but rendering engine-agnostic—interface for generating geographic maps in JavaScript
  3. A jQuery plugin that ties together these two concepts, making it frighteningly easy to make web-native geographic maps

Examples

  • Polymaps
  • ModestMaps
  • Dynamic Polymaps or ModestMaps using LABjs and with feature detection courtesy of Modernizr

Rendering Engines

HTMAPL currently supports two JavaScript map rendering engines: Polymaps and ModestMaps. The library provides "struts" for each rendering engine (one for Polymaps, another for ModestMaps) that expose a common interface to HTMAPL internally.

I've left open the possibility of other "pluggable" interfaces (via the globally accessible jQuery.fn.htmapl.engine variable) and the option to specify different rendering engines on a per-execution chain basis. Conditional rendering engine selection is also possible with LABjs and Modernizr.
